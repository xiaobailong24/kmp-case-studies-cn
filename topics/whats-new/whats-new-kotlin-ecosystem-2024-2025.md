# What's new in the Kotlin ecosystem - 2024/2025

🔥 The last six months have been huge for Kotlin! From major feature releases to stability milestones, here's what we've achieved together:

## Major Achievements

### ✅ Kotlin Language Server Protocol is now in Preview

The Kotlin Language Server Protocol (LSP) provides enhanced IDE support and improved development experience across different editors and IDEs. This preview release marks a significant step toward better tooling integration.

**Benefits:**
* Enhanced IDE support across multiple platforms
* Improved code completion and navigation
* Better integration with various development environments
* Foundation for future tooling improvements

### ✅ Compose Multiplatform for iOS is now Stable

After extensive development and testing, Compose Multiplatform for iOS has reached stability, enabling production-ready multiplatform UI development.

**What this means:**
* Production-ready applications can now be built for iOS using Compose Multiplatform
* Comprehensive API surface for writing beautiful applications
* Minimal migration issues with backward compatibility guarantees
* Performance optimizations and iOS-specific integrations

See the [supported platforms documentation](../overview/supported-platforms.md) for the complete stability overview.

### ✅ New Kotlin Multiplatform plugin for IntelliJ IDEA and Android Studio

We've released an enhanced Kotlin Multiplatform plugin that significantly improves the development experience in both IntelliJ IDEA and Android Studio.

**Key improvements:**
* Better project setup and configuration wizards
* Enhanced navigation between platform-specific code
* Improved debugging capabilities across platforms
* Streamlined development workflow for multiplatform projects

### ✅ Kotlin Rich Errors feature is in development

The Rich Errors feature aims to provide better error reporting and debugging capabilities, making it easier for developers to understand and fix issues in their code.

**Expected benefits:**
* More detailed and contextual error messages
* Better debugging information and stack traces
* Enhanced IDE integration for error highlighting
* Improved developer productivity through clearer diagnostics

### ✅ Kotlin 'Unused return value checker' will be available in Kotlin 2.3

This new static analysis feature helps improve code quality by detecting when function return values are ignored when they shouldn't be.

**Features:**
* Enhanced static analysis for code quality
* Detection of potentially problematic unused return values
* Integration with IDE warnings and suggestions
* Configurable rules for different coding styles

### ✅ Kotlin Notebook is now bundled in IntelliJ IDEA

Kotlin Notebook provides interactive development and data exploration capabilities directly within IntelliJ IDEA.

**Capabilities:**
* Interactive code execution and visualization
* Data science and exploration workflows
* Educational and prototyping use cases
* Seamless integration with IntelliJ IDEA features

### ✅ Junie, the AI coding agent by JetBrains, is now available for Android Studio

Junie brings AI-powered development assistance to Android Studio, helping developers write better code more efficiently.

**AI assistance features:**
* Intelligent code completion and suggestions
* Automated refactoring recommendations
* Code generation assistance
* Context-aware development support
