# Kotlin Multiplatform Plugin for IDEA & Android Studio

> Issue [#48](https://github.com/xiaobailong24/kmp-case-studies-cn/issues/48) · 标签：`articles` `releases` `tools`

## 概述

JetBrains 推出了独立的 Kotlin Multiplatform 插件，为 IntelliJ IDEA 和 Android Studio 提供 KMP 专属开发体验。

### 核心功能

- KMP 项目创建向导
- 多平台模块结构可视化
- expect/actual 导航与补全
- iOS 运行配置与调试
- Compose Multiplatform Preview

### 安装

通过 IDE 插件市场搜索 "Kotlin Multiplatform" 安装。
