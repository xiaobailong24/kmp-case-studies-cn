# Android Gradle Plugin 9.0

> Issue [#72](https://github.com/xiaobailong24/kmp-case-studies-cn/issues/72) · ⚒️ Android Gradle Plugin 9.0

## 概述

AGP 9.0 是 Android 构建工具链的重要升级，配合 Gradle 9.0 使用。

### 关键变化

- 要求 Gradle 9.0+
- 构建性能改进
- 对 Kotlin Multiplatform 项目的改进支持
- 新的 DSL API
