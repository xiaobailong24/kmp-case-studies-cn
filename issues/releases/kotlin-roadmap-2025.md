# Kotlin Roadmap Update 2025 August

> Issue [#32](https://github.com/xiaobailong24/kmp-case-studies-cn/issues/32) · 标签：`documentation` `releases` · 2025-08-25

## 里程碑

2025 年下半年 Kotlin 生态取得了以下重要进展：

- **Kotlin Language Server Protocol** 进入 Preview 阶段
- **Compose Multiplatform for iOS** 正式 Stable
- **Kotlin Multiplatform 插件** 面向 IntelliJ IDEA 和 Android Studio 发布
- **Rich Errors** 特性开发中
- **未使用返回值检查器** 将在 Kotlin 2.3 中推出
- **Kotlin Notebook** 已内置于 IntelliJ IDEA
- **Junie**（JetBrains AI 编程助手）面向 Android Studio 推出
