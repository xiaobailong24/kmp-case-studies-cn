# Firebase in Kotlin Multiplatform

> Issue [#9](https://github.com/xiaobailong24/kmp-case-studies-cn/issues/9) · 标签：`libraries`

## 概述

在 KMP 应用中集成 Firebase，需要处理多平台配置和 CocoaPods 依赖。

### 学习要点

1. 为 KMP 应用配置多环境 Firebase（dev/staging/prod）
2. 在 Kotlin 中使用 CocoaPod iOS 库
3. 从 Swift 调用 Kotlin 代码
4. 构建完整的 Firebase 支持的 KMP 应用

### 常用 Firebase KMP 库

| 库 | 支持平台 |
|----|----------|
| [firebase-kotlin-sdk](https://github.com/GitLiveApp/firebase-kotlin-sdk) | Android, iOS, Web |
| Google 官方 Firebase KMP（开发中） | Android, iOS |
